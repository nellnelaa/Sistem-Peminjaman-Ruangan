Aplikasi web fullstack untuk pengelolaan peminjaman ruangan 

Sistem Peminjaman Ruangan adalah aplikasi berbasis web yang memungkinkan pengguna untuk:
- Melihat daftar ruangan yang tersedia
- Membuat permohonan peminjaman
- Admin menyetujui atau menolak permohonan
- Verifikasi berkas dengan Machine Learning
Tujuan utamanya adalah mengotomasi proses peminjaman ruangan agar tidak manual lagi.
